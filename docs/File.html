Class: File

File

new File(spFile)

Represents a SharePoint File
Parameters:
Name Type Description
spFile SP.File A raw SP.File object
Properties:
Name Type Description
name string Folder name
title string Folder title
serverRelativeUrl string Server relative url
byteLength int File size in bytes
checkoutType string Checked out status of file. "none", "offline", "online".
majorVersion number Major version of the file
minorVersion number Minor version of the file
uri string API url to get the raw SP.Folder object
Example
dao.web.getFile("/sites/mysite/Shared Documents/myFile.docx")
			.then(function(file) { 
				console.log(file.name);
				console.log("Size:" + (file.byteLength / 1000) + "KB");
			});
