lists(listname)

If a list name is passed, an SPScript.List object, otherwise performs a request to get all the site's lists
Parameters:
Name Type Argument Description
listname string <optional>
 If a list name is passed, method is synchronous returning an SPScript.List
Returns:
- SPScript.List object or a Promise that resolves to an Array of lists
Type
List | Promise.<Array.<lists>>
Examples

Option 1: No List Name gets all the lists of a site

dao.lists().then(function(lists) { console.log(lists)});

Option 2: Pass a List Name to get a list object

var list = dao.lists('MyList');
list.getItemById(12).then(function(item) { console.log(item)});
