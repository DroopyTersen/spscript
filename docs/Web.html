Class: Web

Web

new Web(dao)

Represents a SharePoint site
Parameters:
Name Type Description
dao BaseDao Data access object used to make requests.
Properties:
Name Type Description
permissions Permissions allows checking security information of the Web
baseUrl string API relative url (value = "/web")
Example

You access this Web class using the 'web' property of the dao

var dao = new SPScript.RestDao(_spPageContextInfo.webAbsoluteUrl);
dao.web.info().then(function(info) { console.log(info) });

Methods

copyFile(sourceUrl, destinationUrl, requestDigest)

Copies a file
Parameters:
Name Type Argument Description
sourceUrl string The server relative url of the file you want to copy
destinationUrl string The server relative url of the destination
requestDigest string <optional>
 The request digest token used to authorize the request. One will be automatically retrieve if not passed.
Example
var sourceFile = "/sites/mysite/Shared Documents/myfile.docx";
var destination = "/sites/mysite/Restricted Docs/myFile.docx";
dao.web.copyFile(sourceFile, destination).then(function() { console.log("Success") });
